# Privacy Policy for Killo Selection Translator

Last updated: 2026-04-22

Killo Selection Translator is a Chrome extension that provides text selection translation, pronunciation, and local wordbook features.

## What data we process

- Selected text content from web pages (when the user selects text)
- Screenshot text content (only when the user actively triggers screenshot translation)
- Extension settings (such as source/target language preferences)
- Local wordbook entries (including source text, translation, and optional page URL/title context)

## How data is used

- To provide translation, dictionary, pronunciation, and screenshot OCR features requested by the user
- To store user preferences and wordbook entries locally in the browser

## Third-party services

To provide translation and OCR functionality, selected text may be sent to third-party APIs configured by the extension, such as:

- api.mymemory.translated.net
- api.dictionaryapi.dev
- api.ocr.space
- translate.googleapis.com

## Data storage and sharing

- Wordbook and settings are stored locally using Chrome storage
- We do not sell user data
- We do not use user data for advertising
- We do not use user data for creditworthiness or lending decisions

## Data retention and control

Users can clear local wordbook data at any time from the extension settings.
